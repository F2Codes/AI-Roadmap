# AI Roadmap

## 👋 Introduction

Hello everyone!

I'm Matin, a 14-year-old developer from Iran. 💙

Today, I've decided to write this open-source article about Artificial Intelligence (AI).

But why?

As you probably know, AI is everywhere these days. People are talking about it at school, at work, in cafés, on social media—almost everywhere you look.

You might have wondered:

«"What if I learn AI? Could I become an AI engineer someday?"»

The answer is yes—if you're willing to learn consistently and stay curious.

The purpose of this guide is to help beginners understand where to start, what to learn, which resources to use, and how to build a clear roadmap toward becoming an AI engineer.

Whether you're a student, a programmer, or simply someone curious about AI, I hope this guide makes your learning journey easier.
